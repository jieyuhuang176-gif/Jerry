# 主题活动策划Skill - 输入变量

每次使用Skill时，请填写以下变量：

| 变量名称 | 填写内容 |
|----------|----------|
| **活动月份**： | ____________ |
| **活动开始时间**： | ____________ |
| **活动结束时间**： | ____________ |
| **目标人群**： | ____________ |
| **关联课程**： | ____________ |

## 填写说明
- 活动月份：填写数字或中文月份均可（如：6月 / 六月）
- 时间格式：建议使用「YYYY年MM月DD日」格式
- 目标人群：描述要具体，如年龄范围、用户标签等
- 关联课程：填写课程全称，避免简称

## 示例填写
```
活动月份：6月
活动开始时间：2026年6月1日
活动结束时间：2026年6月23日
目标人群：45-70岁超V核心用户（已结营、未学过食养品类的老客户）
关联课程：健康食养基础课
```
