# 主题活动策划Skill - 使用说明

## 适用场景
本Skill适用于为45-70岁中老年用户策划主题活动，特别适合：
- 月度主题活动策划
- 节气/节日活动策划
- 课程引流活动策划
- 用户留存唤醒活动策划

## 使用步骤

### 第一步：准备输入变量
根据「input_variables.md」要求，准备以下信息：
| 变量名称 | 说明 |
|----------|------|
| **活动月份** | 活动所在的月份（1-12月） |
| **活动开始时间** | 活动开始日期 |
| **活动结束时间** | 活动结束日期 |
| **目标人群** | 活动主要面向的人群描述 |
| **关联课程** | 活动需要引流的课程名称 |

### 第二步：调用Skill
将准备好的变量输入，触发Skill生成活动方案。

### 第三步：质量检查
根据「quality_checklist.md」逐条检查输出方案：
- ✅ 无违禁词
- ✅ 语气符合人设
- ✅ 主题与月份强关联
- ✅ 痛点精准

### 第四步：方案调整
根据检查结果，对方案进行必要的调整和优化：
- 修改不合适的文案
- 补充缺失的内容
- 调整活动节奏

### 第五步：落地执行
将优化后的方案交付相关团队执行，并跟踪活动效果。

## 使用示例

**输入：**
```
活动月份：11月
活动开始时间：2026年11月9日
活动结束时间：2026年11月23日
目标人群：45-70岁超V核心用户（已结营、未学过太极品类的老客户）
关联课程：太极体验课
```

**输出：**
生成完整的11月主题活动策划方案，包含活动概述、主题解读、核心玩法、激励机制、执行节奏和行动号召。

## 注意事项
1. 变量填写要准确、具体，以便生成更精准的方案
2. 关联课程植入要自然，避免硬性推销
3. 活动玩法要考虑中老年用户的操作便利性
4. 激励机制要兼顾吸引力和成本控制
