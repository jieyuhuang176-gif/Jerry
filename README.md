# 主题活动策划 Skill

为45-70岁超V核心用户策划温暖、有情感共鸣的主题活动。

## 功能特点

- 🎯 **精准人群定位**：专为中老年用户设计，理解他们的情感需求和生活习惯
- 📅 **月份主题匹配**：自动关联月份、节气，生成贴合时令的活动主题
- 🎨 **多品类适配**：支持书法、太极、瑜伽、摄影、美食等多种课程引流
- 💬 **温暖语气**：亲切关怀的家人般语气，避免生硬营销话术
- ✅ **合规保障**：内置违禁词检查，确保内容合规

## 安装方式

```bash
npx skills add github:your-username/theme-activity-planner
```

## 使用方法

### 输入变量

| 变量名称 | 说明 | 示例 |
|----------|------|------|
| 活动月份 | 活动所在月份（1-12月） | 2月 |
| 活动开始时间 | 活动开始日期 | 2026年2月9日 |
| 活动结束时间 | 活动结束日期 | 2026年2月23日 |
| 目标人群 | 活动面向人群描述 | 45-70岁超V核心用户 |
| 关联课程 | 需要引流的课程名称 | 书法体验课 |

### 调用示例

```bash
skills run theme-activity-planner \
  --活动月份 "2月" \
  --活动开始时间 "2026年2月9日" \
  --活动结束时间 "2026年2月23日" \
  --目标人群 "45-70岁超V核心用户（已结营、未学过书法品类的老客户）" \
  --关联课程 "书法体验课"
```

### 输出

生成完整的超V主题活动全景操作方案，包含：
- 活动概述与背景
- 底层模型建构
- 活动目标测算
- 全周期阶段规划
- 用户洞察与主题定位
- 核心玩法与激励机制
- 私域触达矩阵
- 风险控制与预算

## 支持的课程类型

| 课程类型 | 推荐主题方向 | 拍摄形式 |
|----------|--------------|----------|
| 书法/阅读 | 墨香时光、书韵悠长 | 书法展示、读书角落 |
| 太极/瑜伽 | 动静之间、身体之美 | 运动抓拍、身姿特写 |
| 摄影/短视频 | 光影日记、日常碎片 | 随拍记录、vlog |
| 美食烹饪 | 食光之旅、餐桌上的艺术 | 美食摆盘、烹饪过程 |
| 唱歌/声乐 | 时光乐章、岁月如歌 | 唱歌视频、音乐分享 |
